This is a project for C class taken in Florida Atlantic University.
It is a trading App that provides an imperfect simulation of stock market with specific set of functions
that allow the user to interact (buy/sell/etc), check current prices/balance/portfolio. Everything is saved in files, could be used continuously.

Files needed for compilation:
tradingapp.c
tradingmain.c
common.c
simulator.c
trading.c
tradingapp.h
trading.h
common.h
simulator.h


mystocks.txt must be present for proper work
